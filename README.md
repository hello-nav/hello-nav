## 👩🏻‍💻 Develop & Build

```bash
# Clone project
git clone https://github.com/hello-nav/hello-nav.git

# Change directory to project directory
cd hello-nav

# Installation dependencies
pnpm install

# Develop project
pnpm run dev
```

```bash
# Build project
pnpm run build
```
